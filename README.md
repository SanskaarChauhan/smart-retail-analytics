# Smart Retail Analytics System

An AI-powered retail analytics dashboard built using Python and Streamlit.

## 🚀 Features

- Demand Forecasting (Linear Regression, Random Forest, ARIMA)
- Customer Segmentation (K-Means Clustering)
- Role-Based Access Control (Admin & Employee)
- Admin Approval System for User Access
- Data Upload with Validation Pipeline
- Interactive Dashboard Visualizations

## 🧠 Technologies Used

- Python
- Pandas, NumPy
- Scikit-learn
- Statsmodels
- Streamlit
- Matplotlib

## 🔐 System Design

- Authentication system with hashed passwords
- Admin-controlled user onboarding
- Role-based dashboard access
- Data validation before processing

## 📊 AI Components

- Time series forecasting for revenue prediction
- Customer segmentation using clustering
- Business insights generation

## 🌐 Deployment

Deployed using Streamlit Cloud
